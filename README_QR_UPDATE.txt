UPDATED WEBSITE WITH QR CODE

This version embeds the QR code directly into the Hotel & Event Partnerships / Scan to Book section.

No images folder is required for the QR code; it is embedded inside index.html.

To publish:
1. Open your GitHub repo.
2. Open index.html.
3. Delete everything currently inside.
4. Paste the full contents of this updated index.html.
5. Commit changes.
6. Wait 1-3 minutes.
7. Hard refresh your browser with Ctrl + F5.

Important:
If the QR code ever changes, ask ChatGPT to update the website file again with the new QR code.
